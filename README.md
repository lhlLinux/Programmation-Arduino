## Programmation embarquée (Arduino)

Ce répertoire comporte tous les laboratoires effectués sur un Arduino Uno R3 dans le cadre du cours de capteurs et actionneurs.

Dans chaque répertoire apparaissent plusieurs fichiers .ino correspondant aux différentes étapes d'un même laboratoire. La programmation est effectuée en langage C dans l'IDE d'Arduino.

#### Description des répertoires:
- Lab01: Clignotement DEL
- Lab02: Barographe
- Lab03: Boutons poussoirs
- Lab04: Moteur CC et transistors
- Lab05: Afficheur Adafruit RGB LCD
- Lab06: Servomoteur
- Lab07: Photorésistance
- Lab08: Capteur de température
- Lab09: Entrée analogique et DEL RGB 
- Lab10: Relais

#### Technologies ustilisées:
- Windows 8.1
- Arduino IDE
- Kit Arduino Uno R3
- Langage C
